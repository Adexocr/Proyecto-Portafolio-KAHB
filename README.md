Descripcion del proyecto:
El siguiente proyecto es un proyecto del curso Programacion Web 1 de la carrera de Dise√±o de Aplicaciones Moviles y Web de FundaTEC Costa Rica.
Es proyecto consiste en aplicar el conocimiento adquirido en ciertos conocimientos adquiridos en el curso, para crear un portafolio con informacion relevante sobre el estudiante. 
Es un reto interesante ya que va a permitir utilizar y manipular lo que se ha estudiando en clase, dando una idea mas clara de los conceptos y funcionalidades de HTML y CSS. 

Tecnologias utilizadas:
Se utilizo GitHub para llevar este repositorio con cada cambio y proceso hecho en la creacion de la pagina. De igual manera se utilizo apoyo con herramientas de Inteligencia Artificial
tales como Copilot y ChatGPT para validar y entender como abordar algunos de los requisitos del proyecto. Se utilizaron paginas web para subir imagenes y poderlas utilizar en el proyecto 
como es el caso de la pagina https://es.imgbb.com/ donde gracias a la Inteligencia Artificial, se pudieron crear y subir las imagenes a la pagina para su utilizacion en el proyecto. 

Conceptos aplicados:
Se vieron los conceptos de HTML tales como la semantica y uso correcto de cada etiqueta para iniciar la creacion del espacio y el lenguaje del mismo, (<!DOCTYPE html> <html lang="es">).
Etiquetas relevantes como <header> y <main> basicas para la base del sitio web. En cuanto a CSS se abarcaron los conceptos de Flexbox para distribuir los elementos en filas o columnas y
Z index para entender cual elemento deberia estar por encima de otro. 

Instrucciones de visualizacion:
Se utilizo la extension LIVE SERVER en GitHub para verificar en todo momento como iba luciendo la pagina. Para la visualizacion por parte de otra persona, puede verse dando click derecho
sobre el archivo y abrirlo directamente en algun navegador (FireFox, Chrome). Son dos archivos, un HTML llamado index.html y otro CSS llamado estilos.css.
